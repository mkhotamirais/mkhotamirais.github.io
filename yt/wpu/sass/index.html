wpu sass

  • yang dibahas: variable, mixin, import & uses, operator & built-in function, nested css
  • extension: live sass compiler (glenn marks)
  • buat file index.html dan style.scss, lalu isi dengan syntax scss (semua syntax css berlaku di scss)
  • lihat kebawah vscode ada tombol: Watch Sass, lalu klik sehingga terbentuk file style.css dan style.css.map di folder yang sama
  • Jika ingin dibuatkan di folder tertentu maka bukan settings (ctrl + ,); cari Live Sass Compiler; lalu pada bagian Formats klik edit in settings.json maka akan terbuka settings.json; cari savePath: null, ganti null jadi nama folder misalnya savePath: "style/css" maka hasil compiler style.scss akan terbentuk di folder style/css/style.css
  • tautkan file style.css ke file index.html

selanjutnya lihat saja pada style.scss dan style.css (lihat perubahannya secara real time setiap mengganti file style.scss)
