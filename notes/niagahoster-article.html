Agar website muncul di hasil pencarian google

sumber
  • Jika ingin website muncul di hasil pencarian Google, url website harus ada di google index
    • Google index: database berisi info dan halaman website yang dikumpulkan Google
    • Database itu didapat dari hasil crawling (proses mengikuti berbagai link agar bisa menemukan konten baru) dan indexing (proses menyimpan setiap halaman website ke dalam database agar muncul di hasil pencarian) dengan teknologi bernama spider
    • Googlebot adalah spider milik google
  • Ada dalam index berarti google menyadari website anda tapi tidak secara otomatis menaruh di ranking tertinggi pencarian.
  • Ranking berhubungan dengan SEO jadi masuk index dulu lalu berkompetisi di SEO
  • sebelumnya pahami dulu perbedaan SEO (search engine optimization) dengan SEM (search engine marketing: strategi marketing berbayar untuk meningkatkan visibilitas website di hasil pencarian)
    • Terdapat ikon 'iklan' di hasil pencarian SEM sedangkan SEO tidak ada
    • SEM berbayar SEO gratis
    • SEM bisa memilih target audience
    • SEM hasilnya lebih cepat
    • SEM lebih baik untuk testing SEO tidak
    • SEO baik untuk jangka panjang
    • Berdasarkan studi AWR Potensi CTR SEO lebih tinggi dibanding SEM
      • CTR (Click Through Rate) a/ rasio jumlah klik terhadap jumlah impression. Biasanya klik bersumber dari hasil pencarian organik. Impression a/ jumlah munculnya halaman web di hasil pencarian
      • Menghitung CTR
  • Berikut penjelasan SEO
    • a/ upaya/teknik optimasi tujuannya untuk mendapat trafik tinggi dari hasil pencarian organik sesuai aturan tertentu
    • Tekniknya banyak, sejara umum jenisnya ada dua SEO On page (optimasi berfokus dalam konten) dan SEO Off page (optimasi yang dilakukan di luar konten)
    • adalah teknik/upaya optimasi untuk mendapat ranking teratas di hasil pencarian Google
    • Mengapa google tidak search engine lainnya? karena google search engine favorit pengguna internet lihat
  • Cek apakah website anda sudah masuk google index dengan 'site:websiteanda.com'
  • Cara-cara agar website masuk google index
    • Tambahkan website ke google search console (alat gratis pemilik untuk memonitor kerja website) fungsinya memperlihatkan seberapa sering googlebot crawling ke website anda, memperbaiki error, serta memasukkan sitemap
    • Buat Sitemap XML: daftar dari halaman website (ibarat daftar isi buku) yang ditunjukkan kepada mesin pencari agar website lebih mudah ditemukan sehingga lebih cepat masuk index; ada juga sitemap HTML ditujukan untuk pengunjung website, walaupun tidak berefek langsung kepada google index tapi tetap disarankan membuatnya
    • Buat Robots.txt: untuk memberi tahu mesin pencari halaman mana yang perlu diindeks dan yang tak perlu. pada kondisi tertentu ada halaman yang tidak perlu masuk index google misalnya jika ada konten duplikat
    • Buat Konten yang up to date
    • Perbanyak internal link
    • Buat halaman kategori konten
    • Bangun backlink berkualitas
SEO Strategi SEO Google Indeks Web Crawler (spiders): Tool untuk mengindeks atau mengunduh konten dari internet lalu disimpan ke dalam database mesin pencari. Web Crawler mengumpulkan/mengindeks info apa saja dari artikel gambar video dan lainnya. Tiap search engine punya Web Crawler sendiri: Google Googlebot; Bing Bingbot; Yahoo Slurp bot; DuckDuckGo DuckDuckBot; Baidu Baiduspider (china); Yandex Yandex Bot (mesin pencari Russia); Sogou Sogou Spider (dari china); Exalead Exabot; Amazon Alexa Crawler. Google penguasa pangsa pasar mesin pencari maka pastikan agar website terindex di Googlebot. Web Crawler mulai pekerjaannya berdasar daftar link halaman yang sudah ia kenal sebelumnya dari sitemap suatu website. dari daftar link sitemap tersebut, ia akan menemukan links lain yang tersebar di dalamnya lalu melakukan crawling ke links yang baru saja ditemukan itu, proses ini terjadi berulang tak henti. Namun dalam crawling ada 3 pertimbangan: 1) seberapa penting dan relevan suatu halaman, banyak pengunjung; 2) kunjungan rutin; 3) menuruti keinginan robots.txt Fungsi Web Crawler: utamanya mengindeks konten internet, fungsi lain: 1) bandingkan harga; 2) data untuk tools analisis; 3) data untuk statistik; 4) Pengaruh web crawler kepada SEO. Bila web crawler tidak mengindeks website anda, maka website tidak akan muncul di hasil pencarian Sitemap dengan xml-sitemap.com Setelah Web Hosting dan Domain perhatikan back-end nya salah satunya Sitemap. Diantara kontribusinya mendapatkan peringkat tinggi di SERP juga strategi SEO. Sitemap peta yang menjabarkan setiap halaman website, dibuat dalam format XML (Extensible Markup Language) dan digunakan AI untuk mempermudah index website. Fungsi utama Sitemap bagi website untuk membantu maksimalkan strategi SEO. Sitemap berisi frekuensi update, keterkaitan satu halaman dengan halaman lainnya dan landing page serta tingkat kepentingan halaman tersebut. Bahkan memberi tahu mesin pencari bahwa website tidak memiliki konten duplikat yang merupakan masalah serius dalam SEO. Cara membuat: langkah pertama mencari tahu tata letak jalur website, semakin simpel semakin mudah crawler menentukan peringkat di SERP. Misalnya halaman yang dalamnya hingga 3 kali klik dari halaman root, akan lebih sulit untuk diberi peringkat. Cara membuat sitemap Manual: buat file dengan ekstensi .xml lalu isi kode relevan. Lalu validasi menggunakan sitemap validator https://www.xml-sitemaps.com/validate-xml-sitemap.html agar semua upaya optimasi tidak sia sia. Cara membuat sitemap Otomatis: dengan generator https://www.xml-sitemaps.com/ apabila di wordpress bisa denga plugin https://yoast.com/wordpress/plugins/seo/ Cara submit sitemap: ke mesin pencari. Pertama, upload sitemap ke website lalu dapatkan alamat URL untuk proses pengiriman. Untuk submit ke google gunakan https://search.google.com/search-console/welcome ke bing gunakan https://www.bing.com/webmasters/about untuk DuckDuckGo tidak perlu karena menggunakan informasi dari bing dan google
